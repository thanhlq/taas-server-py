## Quick Command Reference

```bash
# 1. Basic test (easiest)
./scripts/run_websocket_test.sh

# 2. Python test (automated)
python scripts/test_websocket.py

# 3. HTML test (visual)
open scripts/test_websocket.html

# 4. Stress test
python scripts/test_websocket.py --test-type stress --num-messages 100

# 5. Custom server
./scripts/run_websocket_test.sh -h production.com -p 443

# 6. Multiple concurrent users (run in separate terminals)
python scripts/test_websocket.py --user-uuid alice &
python scripts/test_websocket.py --user-uuid bob &
python scripts/test_websocket.py --user-uuid charlie &
```
